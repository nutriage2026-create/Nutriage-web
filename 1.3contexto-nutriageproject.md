## Stack tecnológico
- **100% HTML + CSS + JS** en un solo archivo (sin frameworks, sin backend)
- **localStorage** como base de datos del navegador (temporal, para beta)
- **Formulario propio** dentro del código (reemplazó Tally para guardar datos internamente)
- **Sin servidor, sin base de datos externa** — todo corre en el navegador
- Referencia original de formulario Tally: https://tally.so/r/b589L6

## Arquitectura del sistema — 3 vistas

### 1. Vista Paciente (pública)
Flujo de 4 pasos en orden:
1. **Calendario** → elige fecha y hora disponible
2. **Formulario** → 33 campos clínicos (ver lista abajo)
3. **Confirmación** → resumen de reserva
4. **Pago** → datos de transferencia + subir comprobante

### 2. Vista Nutricionista (contraseña: fernanda2026)
- Citas del día (nombre, hora, motivo)
- Calendario editable de disponibilidad (Fernanda activa/desactiva horarios por día)
- Registros de todos los pacientes desplegables
- Botón "Ver ficha clínica completa" por paciente (abre modal con TODA la info)
- Ve el comprobante de pago de cada paciente

### 3. Vista Administrador (contraseña: admin2026)
- Estadísticas: total pacientes, consultas hoy, ingresos esperados, formularios con comprobante
- Tabla completa de pacientes con botón "Ver ficha"
- Modal con ficha completa: datos personales, historial médico, hábitos, registro 24h, comprobante
- Tabla de estadísticas de Fernanda: sesiones e ingresos diario/semanal/mensual/anual

## Campos del formulario (33 campos)
**Datos personales:** nombre, email, teléfono, edad, ocupación, peso (kg), talla (metros)

**Historial médico:** motivo de consulta, enfermedades diagnosticadas, lesiones recientes, antecedentes familiares, cirugías

**Estilo de vida:** convivencia/quién cocina, actividad física, horario laboral, horario de sueño

**Hábitos alimentarios:** alimentos que no le gustan, almuerzo frecuente, horarios de comida, alergias/intolerancias, suplementos

**Consumo:** alcohol, tabaco/drogas

**Digestión:** síntomas gastrointestinales

**Frecuencia semanal:** cereales, legumbres, pescado, fruta, verduras, dulces, lácteos

**Registro 24h:** descripción libre con instrucciones y ejemplo
